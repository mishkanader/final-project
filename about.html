Information about what was used to create this site.

About Permanent Addiction


Bootstrap:

  • "Bootstrap, a sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development." - Bootstrap
  • We used this to uniformly create the layout of the website.
  • We also used this to make it responsive!
  • Other things included (but not limited to): navigation bar, footer, forms, buttons.

Cascading Style Sheets (CSS):

  • "A style sheet language used for describing the presentation of a document written in a markup language." - CSS
  • We used this to uniformly style the layout of the website.
  • We changed the color of the buttons and navbar.
  • Control what each part of the website will look like.

HTML Comment Box:

  • "HTML Comment Box widget for your website. Copy the HTML code and it just works!" - HTML Comment Box
  • Easy solution for users to be able to interact with one another.
  • They can ask questions and hopefully get help from other users.

jQuery:

  • "jQuery: The Write Less, Do More, JavaScript Library." - jQuery
  • This was necessary for Bootstrap's JavaScript plugins.
  • This is what makes things responsive.
  • Other things included (but not limited to): navigation bar, footer, forms, buttons.

MySQL/phpMyAdmin:

  • "It is the most popular Open Source Relational SQL database management system." - MySQL
  • "A tool written in PHP intended to handle the administration of MySQL over the WWW." - phpMyAdmin
  • We used this to store information.
  • The register and login "pages" send and verifies a user's input and information (respectively) to the database.

PHP:

  • "A server scripting language, and a powerful tool for making dynamic and interactive Web pages ." - PHP
  • This was used to connect the forms to the database so that we cna store the information and retrieve it when needed (i.e. register and login).
